## 1. レイアウトパターン（2種類）

### パターンA: 2ショット（クローズアップ＋全身）
**使用条件**: 身長15cm以上の細身・スリムなフィギュア

```
┌─────────────┬───────────┬──┐
│             │           │  │
│  クローズ   │  全身     │ ●│
│  アップ     │  （小）   │ ｜│
│  （大）     │           │28│
│  左55〜60%  │ 右35〜40% │cm│
└─────────────┴───────────┴──┘
```
- クローズアップ: 左寄り、大きく（上50〜55%をクロップ）
- 全身: 右側、キャンバス高さの85〜92%
- サイズバッジ: 全身の右端に沿って配置

**事例**: 14034(21cm), 14036(18cm), 14037(19cm), 14041(28cm), 14043(21cm), 14046(19cm), 14047(26cm), 14052(17cm)

---

### パターンB: 1ショット（全身のみ・中央配置）
**使用条件**: チビ・丸いフィギュア OR 15cm以下の小型

```
┌─────────────────────────┬──┐
│                         │  │
│    フィギュア全体        │ ●│
│    （中央・大きく）      │ ｜│
│                         │15│
│                         │cm│
└─────────────────────────┴──┘
```
- フィギュア: キャンバス95%の高さ、水平中央配置
- サイズバッジ: 右端固定

**事例**: 14045(ルフィ15cm・チビ), 6550B(カカシ10cm), 6550D(イタチ11cm)

## 3. サイズバッジのスタイル（2種類）

### スタイルA: 塗りつぶしピル型（Filled）
**使用条件**: 可愛い・チビ・明るい世界観
- 背景: ダークネイビー（#12123A 不透明）
- 外枠: なし or 細い白線
- 事例: 14034, 14045, 14046

### スタイルB: アウトラインピル型（Outlined）
**使用条件**: クール・リアル・シリアス系
- 背景: 半透明ネイビー（不透明度50〜60%）
- 外枠: 白 2〜3px
- ドット: 白（不透明）
- 事例: 14036, 14039, 14043, 14047, 14052, 6550B, 6550D

---

## 4. 共通デザインルール

- **キャンバスサイズ**: 1200×1200px（正方形）
- **出力フォーマット**: WebP（quality=90）
- **サイズバッジのドットライン**: 全身フィギュアの実際の頭頂〜足元座標に合わせる
- **クローズアップの余白**: 上下左右ブリード（余白ゼロ）
- **レイヤー順序**: 背景 → クローズアップ → 全身 → バッジ（全身が最前面）
- **背景**: キャラ世界観に合わせた AI 生成（tilt-shift/ミニチュア感は不要なものもある）

---

## 5. 注目ポイント（品質のカギ）

1. **背景とフィギュアの世界観統一**が最重要（ミスマッチは品質を大きく下げる）
2. クローズアップと全身で**同じフィギュア・同じアングル種**を使う
3. 14039（白ひげ）のようにリアル写真風背景も有効（高品質感が増す）
4. 14043（VTuber）のように**台座込み**で全身を見せるパターンもある
5. チビキャラは1ショットが断然きれい（2ショットにすると頭が大きすぎる）
