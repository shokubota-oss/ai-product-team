## 1. レイアウトパターン（4種類）

### パターンA: 2ショット（クローズアップ＋全身）
**使用条件**: 高さ/幅 ≥ 1.8 のスリム・細身フィギュア

```
┌─────────────┬───────────┬──┐
│  クローズ   │  全身     │ ●│
│  アップ     │  （小）   │ ｜│
│  （大）     │           │28│
│  左55〜60%  │ 右35〜40% │cm│
└─────────────┴───────────┴──┘
```
**事例**: 14034(21cm), 14036(18cm), 14037(19cm), 14041(28cm), 14043(21cm), 14046(19cm), 14047(26cm), 14052(17cm)

---

### パターンB: 1ショット（単体・中央配置）
**使用条件**: 高さ/幅 < 1.8 のチビ・丸いフィギュア、ぬいぐるみ、クリーチャー

```
┌─────────────────────────┬──┐
│    フィギュア全体        │ ●│
│    （中央・大きく）      │ ｜│
│    キャンバス高さ95%     │12│
│                         │cm│
└─────────────────────────┴──┘
```
**事例**: 14045(ルフィ15cm), 6550B(カカシ10cm), 14035(ジョージ16cm), 14040(ジラーチ39cm), 14042(ピカチュウ12cm), 14049(モンスター11cm), 14053A(マイメロ11cm), 14053B(クロミ12cm), 6547C(ベイマックス5cm), 6549A/C(VTuberぬいぐるみ12cm), 6551C(ポケモンぬいぐるみ16cm)

---

### パターンC: 2個並び（ON/OFF比較）
**使用条件**: 発光・ギミック商品（ライトOFF＋ライトON の状態比較）

```
┌──────────────┬──────────────┬──┐
│   状態①     │   状態②     │ ●│
│   (OFF等)    │   (ON等)     │ ｜│
│              │              │10│
│  左に大きく  │  右に斜め    │cm│
└──────────────┴──────────────┴──┘
```
- 左下に **電球アイコン**（💡）で発光機能を示す
- バッジは黒ピル型（filled_black）
**事例**: 14024(シナモロール光るランプ10cm)

---

### パターンD: グッズ2ショット（正面＋別角度）
**使用条件**: バッグ・グッズ類（フィギュアではない）

```
┌─────────────┬───────────┬──┐
│  正面（大） │  別角度   │ ●│
│             │  （小）   │ ｜│
│             │           │26│
│             │           │cm│
└─────────────┴───────────┴──┘
```
- 速度感のある背景（speed lines）が映える
**事例**: 14029B(ポケモンバッグ26cm)

## 3. サイズバッジ（スタイル3種 × ライン2種）

### バッジ形状スタイル

| スタイルキー | 外観 | 使用条件 | 事例 |
|---|---|---|---|
| `filled` | ダークネイビー塗りつぶし | アクション系・バトル系 | 14034, 14045, 14046 |
| `outlined` | 半透明＋白アウトライン | クール・リアル・シリアス系 | 14036, 14039, 14047 |
| `filled_black` | 黒塗りつぶし（やや細長） | 可愛い・ぬいぐるみ・サンリオ系 | 14024, 14053A/B, 6549A/C, 6551C |

### ラインスタイル

| スタイル | 外観 | 使用条件 |
|---|---|---|
| `dotted` | 小さな白ドットの連続 | アクション・クール系（outlined badge と組み合わせ多） |
| `solid` | 細い実線 | 可愛い・ぬいぐるみ系（filled_black badge と組み合わせ多） |

**組み合わせ傾向**:
- `filled_black` ＋ `solid` → サンリオ・ぬいぐるみ・ポケモン可愛い系
- `outlined` ＋ `dotted` → アニメ・アクション・クール系
- `filled` ＋ `dotted` → アクション・バトル系

## 4. 特殊要素

### 発光アイコン（💡）
- 光るギミックのある商品にのみ追加
- 左下に紫の円形アイコンで配置
- 事例: 14024(シナモロール発光ランプ)

### 商品タイプ別判定フロー

```
商品タイプ判定
├── グッズ（バッグ等） → パターンD（speed_lines背景）
├── 発光ギミック → パターンC（電球アイコン付き）
├── ぬいぐるみ → パターンB (filled_black + solid)
└── フィギュア
    ├── 高さ/幅 ≥ 1.8 → パターンA (2ショット)
    └── 高さ/幅 < 1.8 → パターンB (1ショット)
```

## 5. 共通デザインルール

- **キャンバスサイズ**: 1200×1200px（正方形）
- **出力フォーマット**: WebP（quality=90）
- **バッジのドット/ライン**: 全身の実際の頭頂〜足元座標に合わせる
- **クローズアップ余白**: 上下左右ブリード（余白ゼロ）
- **レイヤー順序**: 背景 → クローズアップ → 全身 → バッジ

---

## 6. 品質のカギ（全24枚から）

1. **背景×世界観の一致**が最重要（ベイマックス→テック青、マイメロ→ピンク・ハート等）
2. **バッジ＋ライン**の組み合わせで世界観が変わる
3. ぬいぐるみは素材感が映える背景（ステージ・パステル）を使う
4. 5cmの極小商品でも大きめバッジで視認性を確保（6547C）
5. 同シリーズ（14053A/B, 6549A/C）は背景カラーだけ変えてキャラを差別化
6. セット商品（ピカチュウ＋イーブイ）は台座ごと1ショットで統一感を出す
