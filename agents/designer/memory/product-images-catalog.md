## ファイル命名規則

### 基本形式
```
{商品ID}.png          例: 10000.png（バリエーションなし）
{商品ID}{variant}.png 例: 10000A.png, 10000B.png（バリエーションあり）
```

### バリエーション別件数
| バリエーション | 件数 | 用途 |

## 画像使用ガイドライン

1. **使用前確認**: 上記「使用不可」「登録保留」のIDは使用しない
2. **優先順位**: A バリエーション（メイン）→ B（サブ）→ C以降の順で使用
3. **フォーマット**: WebPが利用可能な場合はWebPを優先（ファイルサイズ小）
4. **アクセス方法**: Google Driveフォルダから直接ダウンロード、またはZIPを展開して使用
