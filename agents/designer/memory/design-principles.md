# デザイン原則・ベストプラクティス

> 収集日: 2026-05-28 | Sources: WebSearch（e-commerce image best practices, WCAG 2.1）

---

## 商品画像のベストプラクティス

### 解像度・サイズ
- **最小解像度**: 2000×2000px（ズーム機能のため）。理想は3000×3000px / 300dpi
- **推奨解像度**: 2048×2048px（Shopify標準）、アスペクト比 1:1
- **ファイルサイズ上限**: 1枚あたり500KB以下（ページ読み込み3秒以内を確保）
- **フォーマット優先順**: WebP → AVIF → JPEG（WebPはJPEG比2.3倍速く読み込み）

### アスペクト比
- **商品一覧・詳細ページ**: 1:1（正方形）が業界標準（Amazon・Shopify・WooCommerce）
- **モバイル・SNSコマース**: 2:3 または 4:5（縦長）でスクロール画面を最大活用
- **ライフスタイル・バナー**: 3:2 または 4:3（横長）
- **一貫性の効果**: アスペクト比統一でCore Web Vitals（CLS）最大18%改善、LCP最大38%改善

### 背景・構図
- **メイン画像の背景**: 純白（RGB 255,255,255）— Amazon必須要件
- **商品占有率**: フレームの**85%以上**を商品が占めること
- **推奨アングル数**: 白背景3〜5枚（正面・斜め45°・側面・背面）
- **コンバージョン効果**: 高品質画像は低品質比でCVR **+94%**。不十分な画像で **30%** の購入離脱

### SEO・テキスト
- **alt text**: 10〜12語で説明的かつ簡潔に記述（キーワードを自然に含める）
- **ファイル名**: ハイフン区切りの説明的命名（例: `red-cotton-tshirt-front.webp`）

### 実装
- **Lazy loading**: スクロール領域外はすべて遅延読み込み
- **srcset属性**: 端末・解像度に応じた複数サイズ配信（Retina対応）

## アクセシビリティ基準（WCAG 2.1）

### 画像のalt text（達成基準 1.1.1 / Level A）
- **意味のある画像**: 目的を果たす同等のテキスト代替が必須
- **装飾的な画像**: `alt=""` でスクリーンリーダーにスキップさせる
- **商品画像のalt text例**: `alt="ネイビー コットン Vネック Tシャツ 前面"`
- **文字数目安**: 140文字以内
- **WebAIM調査**: altテキスト不備はWCAG違反の最多項目（ホームページの**58%**に存在）

### コントラスト比（達成基準 1.4.3 / Level AA）

| 対象 | Level AA 最小値 | Level AAA 目標値 |
|---|---|---|
| 通常テキスト（18pt未満） | **4.5:1** | 7:1 |
| 大テキスト（18pt以上・14pt太字以上） | **3:1** | 4.5:1 |
| UIコンポーネント・グラフィクス | **3:1** | — |

> 商品画像上にテキストを重ねる場合（価格・ラベル等）は必ず上記比率を満たすこと。

### タップターゲットサイズ
- **WCAG 2.2 SC 2.5.8（Level AA）**: 最小 **24×24 CSS px**
- **Apple HIG 推奨**: 44×44px
- **Google Material Design 推奨**: 48×48px
- **実装指針**: 主要CTA（カート・購入）は44×44px以上。ターゲット間の余白は8px以上

## モバイルファーストデザイン原則

### タイポグラフィ
- **本文フォントサイズ**: 16px以上（iOSの自動ズームを防ぐ最低値）
- **行間（line-height）**: 1.4〜1.5（小画面での可読性）
- 価格・CTAは16px以上でスケール

### パフォーマンス目標
- **ページ読み込み**: 3秒以内（超過で直帰率急増）
- **LCP（Largest Contentful Paint）**: 2.5秒以内
- **読み込み1秒遅延**: CVR 最大 **-7%**

### レイアウト
- **商品グリッド（モバイル）**: 2列グリッドが標準
- **スクロール方向**: 縦スクロール優先
- **折り畳み前（Above the fold）**: 価格・CTA・主要商品画像をスクロールなしで表示

### 画像配信
- `srcset` で端末幅に応じた適切解像度を配信
- Retinaディスプレイ対応: 表示サイズの**2倍解像度**アセットを用意
- 折り畳み前の画像: `loading="eager"` / 折り畳み後: `loading="lazy"`
